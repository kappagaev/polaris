# Polaris

Polaris is calendar managing cli

## Format

```markdown
- 11:00 - 15:40 Title | -m -c red -p [string, foo@bar.com, vlad]
description line 1  
description line 2


---

#foo #baz
- 16:00 Title 2 | -c green
  description line 1

---
#bar
- 18:00 - 20:00 Title 3
```
